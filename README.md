# dotfiles

## Description

設定ファイルとか入れとく

## Usage

[make.sh](/make.sh) TeXビルド用

[setup.sh](/setup.sh) ArchLinuxの環境構築スクリプト

[dotfileslink.sh](/dotfileslink.sh) 諸々の環境を自動構築してくれるスクリプト

## Installation

    $ git clone https://github.com/Heppokoyuki/dotfiles.git
    $ ./dotfileslink.sh
